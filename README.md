## Setup

# Step 1

Install packages

```npm install```

# Step 2 

Edit .env file : 

```
db_uri="YOUR_MONGODB_URI"

key="YOUR_CHOICE"

user="GMAIL_ADDRESS"

pass="EMAIL_PASSWORD"

PORT=80
```
# Step 3

Edit all the localhost in the code to the ip you want.
